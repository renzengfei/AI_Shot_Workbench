# 视频分析完整流程

这个工作流指导完成从上传视频到导出分析结果的完整过程。

## Step 1: 上传并自动切分

1. 打开前端界面 http://localhost:3000
2. 点击"上传视频"按钮
3. 选择本地视频文件
4. 等待 AI 自动场景检测完成
5. 在时间轴上查看切分点

## Step 2: 人工微调(可选)

1. 在时间轴上手动添加/删除切点
2. 调整镜头边界
3. 隐藏不需要的镜头

## Step 3: 导出项目

1. 点击"导出项目"按钮
2. 输入项目名称
3. 系统生成:
   - `项目报告.md` - Markdown 格式报告
   - `images/` - 每个镜头的首帧图
   - `videos/` - 每个镜头的视频片段
   - `项目数据.json` - 结构化数据

## Step 4: AI Studio 分析(未来功能)

1. 将导出的数据上传到 AI Studio
2. 使用 Gemini 3 进行骨架分析
3. 导入 JSON 结果回工作台
